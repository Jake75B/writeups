
### Honeypots

- Attract attackers 
- Attacker is likely a machine, honeypots
	- Honeypot is a virtual world that isn't part of the companies production
	- Battle to discern real from fake; As attackers get better at identifying honeypots, the complexity and intelligence of the pot is increased.

### Honeynets

- Includes more then one device:
	- Servers, workstations, routers, switches, firewalls
	- One or more honeypot

### Honeyfiles

- Create files with fake information
	- Something bright or shiny I.e. passwords.txt
	- Alert sent if file is accessed
	- virtual bear trap

### Honeytokens

- Traceable data added to honeynet
	- API credentials, database records, browser cookies, webpage pixels for example
	- fake email addresses, monitor internet to see who posts it
